# Projeto-Linguagens-Formais-e-Automatos
Um projeto acadêmico realizado com o intuito de compreender a transformação de autômatos não determinísticos em autômatos determinísticos, deixando de lado o movimento vazio.
Projeto feito inteiramente em C, praticando a leitura e escrita de arquivos, alocação dinamica de memória e lógica de ponteiros.
